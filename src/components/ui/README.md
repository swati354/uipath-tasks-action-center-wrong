# UI Components

This directory contains shadcn/ui components that are pre-installed in the template.

All components from shadcn/ui are available including:
- Badge
- Button
- Card (CardHeader, CardTitle, CardDescription, CardContent, CardFooter)
- Progress
- Dialog
- Select
- Table
- Tabs
- And many more...

Import them like:
```typescript
import { Button } from './ui/button';
import { Card, CardHeader, CardTitle } from './ui/card';
```

These components are automatically available - no need to create them manually.
